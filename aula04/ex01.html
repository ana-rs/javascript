Sinal de "=" no JS significa "recebe"

Para definir variáveis utilizamos "var"

  • var s1 = "Java Script"
  • var s2 = 'JS'
  • var s3 = `Curso de Java Script `

O nome de cada variável chamamos de identificadores.

  • Os identificadores podem começar com letra, $ ou _
  • Não podem começar com números
  • É possível usar letras ou números
  • É possível utilizar acentros e simbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas
