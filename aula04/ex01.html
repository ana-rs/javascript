Anotações sobre JS

Sinal de "=" no JS significa "recebe"

Para definir variáveis utilizamos "var"

  • var s1 = "Java Script"
  • var s2 = 'JS'
  • var s3 = `Curso de Java Script `

O nome de cada variável chamamos de identificadores.

  • Os identificadores podem começar com letra, $ ou _
  • Não podem começar com números
  • É possível usar letras ou números
  • É possível utilizar acentros e simbolos
  • Não podem conter espaços
  • Não podem ser palavras reservadas
  • MAIUSCULAS e minusculas fazem diferença
  • Escolha nomes coerentes para as variáveis

    number
  • 5
  • 18
  • -12
  • 0.5
  • -15.9
  • 3.14
  • 8.0
    boolean
  • true
  • false
    string
  • "Google"
  • 'Java Script'
  • `Maria`

O "+" serve para adição e concatenação


Formatando Strings

var.length - quantos caracteres a string tem
var.toUpperCase() - tudo para MAIUSCULAS
var.toLowerCase() - tudo para minusculas

Formatando números

var n1 = 1543.5
n1 n1.toFixed(2) - exibe as duas casas decimais n1.toLocaleString('pt-br' , {style: 'curency', curency: 'BRL'}) - para exibir em moeda
